# 7. Database Design

# Таблица users

```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    full_name VARCHAR(255),
    email VARCHAR(255) UNIQUE,
    password_hash TEXT,
    role VARCHAR(50),
    created_at TIMESTAMP DEFAULT NOW()
```

```sql
CREATE TABLE services (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255),
    code VARCHAR(50),
    priority INTEGER DEFAULT 0,
    is_active BOOLEAN DEFAULT TRUE
);
```

---

# Таблица windows

```sql
CREATE TABLE windows (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255),
    status VARCHAR(50),
    current_operator_id INTEGER
);
```

---

# Таблица tickets

```sql
CREATE TABLE tickets (
    id SERIAL PRIMARY KEY,
    ticket_number VARCHAR(50),
    queue_number INTEGER,
    status VARCHAR(50),
    service_id INTEGER,
    window_id INTEGER,
    operator_id INTEGER,
    full_name VARCHAR(255),
    iin VARCHAR(12),
    phone VARCHAR(20),
    priority INTEGER DEFAULT 0,
    estimated_wait INTEGER,
    created_at TIMESTAMP DEFAULT NOW(),
    called_at TIMESTAMP,
    completed_at TIMESTAMP
);
```

---

```sql
CREATE TABLE queue_logs (
```

# Основная логика очереди

## Создание талона

После регистрации:
1. создается ticket;
2. определяется service;
3. рассчитывается очередь;
4. генерируется ticket_number;
5. рассчитывается estimated_wait.

---

## Вызов талона

При вызове:
- выбирается WAITING ticket;
- учитывается priority;
- обновляется статус CALLED;
- отправляется WebSocket event.

---

## Завершение обслуживания

После завершения:
- статус COMPLETED;
- фиксируется время;
- обновляется аналитика.

---

## Пропуск талона

Если талон пропущен:
- статус SKIPPED;
- повторный вызов через configurable timeout.

# Ticket APIs

## POST /tickets

Создание талона.

Request:

```json
{
  "full_name": "Иванов Иван",
  "iin": "123456789012",
  "phone": "+77001234567",
  "service_id": 1
}
```

Response:

```json
{
  "ticket_number": "A-101",
  "status": "WAITING",
  "estimated_wait": 15
}
```

---

## GET /tickets/{id}

Получение информации о талоне.

---

## POST /tickets/{id}/cancel

Отмена талона.

---

# Operator APIs
